\begin{abstract}
Autonomous inspection of large orbital structures requires planning algorithms
that jointly account for relative orbital dynamics, sensor visibility, target
coverage, collision avoidance, and fuel expenditure. This paper presents
OrbInspect, a ROS~2-based simulation and planning framework for external
inspection of space-station-scale structures, and formulates a coverage-
constrained trajectory planning problem in the local-vertical/local-horizontal
frame. The proposed planning workflow first samples inspection targets on a
three-dimensional station model, constructs safe candidate viewpoints, solves an
offline coverage-aware viewpoint ordering problem, and then generates a
dynamically feasible trajectory under Clohessy--Wiltshire relative dynamics.
The resulting trajectory is validated in ROS~2 with a Gazebo Harmonic
visualization using a NASA International Space Station model. The current draft
defines the formal problem, mission assumptions, constraints, and algorithmic
evaluation protocol required before claiming full-coverage experimental results.
\end{abstract}
